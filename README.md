- [x] bootstrap

### 用于学习以下知识
> * 路由
> * redux
> * ajax
> * 单元测试

### 使用方法
```
npm i
npm start
```

### 运行单元测试
```
npm test
```

### 你也可以使用这个webpack进行项目开发，生产环境使用方法，生成的dist目录就是你要给后端的目录
```
npm run build
```
